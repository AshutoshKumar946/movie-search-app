# 🎬 Movie Review App

A web application that allows users to search for movies and view detailed information using the OMDb API. Built with vanilla JavaScript, focusing on API integration and dynamic DOM manipulation.

---

## 🚀 Features
- Search movies in real-time  
- Display movie list with posters and titles  
- Fetch and show detailed movie information  
- Responsive UI for different screen sizes  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **API:** OMDb API  
- **Concepts:** Fetch API, DOM Manipulation, Event Handling  
- **Version Control:** Git, GitHub
